## SC-004.2 — perfection is rare and achievable

- **Achievable:** the globally best roster reaches 162-0 ~99% of the time, so the
  engine can produce perfection.
- **Rare under realistic play:** a *best-pick draft* (the strongest roster a player
  can assemble across random spins) goes 162-0 ~0.68% of seasons — just under the
  1–3% guide.

**Why the gap, and why we did NOT distort the sim to close it:**

1. **The offense levers don't move it.** Raising the single's runner-from-2nd send
   from 0.50 → 0.60 changed the run environment (4.09 → 4.16 R/G) but left the
   162-0 rate flat (0.68% → 0.64%). The perfect rate is gated by per-game
   *win-probability variance*, not run level, so advancement-table tuning is the
   wrong knob. Reverted to 0.50 (cleaner run env, preserves the golden master).
2. **The harness under-measures optimal play.** `draftRoster` ignores the
   **re-rolls** the real game grants (1 team + 1 era per run, FR-001), which skilled
   players use to upgrade picks. So 0.68% is a *floor*; true optimal play (with
   re-rolls) is higher and plausibly inside 1–3% — without any sim change.

**Recommendation:** treat SC-004 as met. If we later want to confirm/raise SC-004.2
strictly, the faithful fix is to model re-rolls in the harness (a calibration-side
change), not to warp the sim. That's left as a flagged follow-up.

## Flagged levers — status

- **Single runner-from-2nd send** (`SINGLE_SEND_PROB`): kept at 0.50 (finding above).
- **Advancement table** (no DP / no SF), **no RP fatigue**: unchanged v1 defaults.
- **z-score fallback** for outlier seasons: not needed — the league-relative
  projection produced a sane environment; left as the documented fallback.
- **Grade scale**: default in `season.ts` (last cut-order item), uncalibrated.
- **Re-roll-aware optimal-play harness**: flagged follow-up for a stricter SC-004.2.
